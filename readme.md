# Dragons of Mugloar #

### Setup:

1. Clone repository
```
git clone https://github.com/gytis88/dragons-of-mugloar.git && cd ./dragons-of-mugloar
```
2. Install required dependencies
```
npm install
```
3. Run solution or tests
```
npm start || npm test
```

 http://www.dragonsofmugloar.com/api
